# ZenusUtils Support

This repository is only used for bug reports, suggestions and support for ZenusUtils.

The source code is currently private and is not included in this repository.

## Modrinth

ZenusUtils is available on Modrinth.

## Bug Reports

Use the Issues tab to report bugs.

Please include:
- Minecraft version
- Fabric Loader version
- ZenusUtils version
- Client used, if any: Minecraft, Lunar Client or Feather Client
- What happened
- How to reproduce the issue
- Logs or screenshots, if possible

## Suggestions

Feature requests and suggestions are welcome in the Issues tab.
