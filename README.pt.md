# Suporte ZenusUtils

Este repositório é utilizado apenas para reportar bugs, sugestões e suporte para ZenusUtils.

O código-fonte é atualmente privado e não está incluído neste repositório.

## Modrinth

ZenusUtils está disponível no Modrinth.

## Reportar Bugs

Use a aba Issues para reportar bugs.

Por favor inclua:
- Versão do Minecraft
- Versão do Fabric Loader
- Versão do ZenusUtils
- Cliente utilizado, se aplicável: Minecraft, Lunar Client ou Feather Client
- O que aconteceu
- Como reproduzir o problema
- Logs ou screenshots, se possível

## Sugestões

Sugestões de funcionalidades e melhorias são bem-vindas na aba Issues.
